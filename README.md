# Spacelift Migration Kit

[![CI](https://github.com/spacelift-io/spacelift-migration-kit/actions/workflows/ci.yml/badge.svg)](https://github.com/spacelift-io/spacelift-migration-kit/actions/workflows/ci.yml)

The Spacelift Migration Kit (SMK) helps customers migrate their infrastructure management from their current solution to
Spacelift with minimal friction.

## Usage

For instructions on using this tool, see [our documentation](https://docs.spacelift.io/product/migrating-to-spacelift).

## Getting Support

To report bugs or request features, [open an issue](https://github.com/spacelift-io/spacelift-migration-kit/issues) on
this repository. For help using the tool, contact our Support team.
